## 📖 项目简介

**精神药物 Wiki** 是一个专注于精神活性物质安全使用的知识库，旨在通过科学、专业的信息帮助用户了解药物风险、减少伤害。

本项目提供：
- 🔬 **药物百科**：详细的药物信息、作用机制、风险评估
- 📝 **用户报告**：152 篇真实案例报告（RP-0 至 RP-151）
- 🏥 **紧急处理**：过量急救、戒断症状管理
- 💚 **康复指南**：戒断支持、心理恢复、重建生活

## ✨ 核心特性

- **减害理念**：基于科学的风险降低策略
- **真实案例**：用户匿名分享的经验报告
- **处方生成器**：交互式工具帮助理解药物信息
- **分类导航**：药物、联用、报告、恢复四大模块
- **移动友好**：响应式设计，支持多端访问
- **快速搜索**：本地全文搜索功能
